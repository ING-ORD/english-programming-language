Information Technology

Problem

Novice IT specialists do not know which programming language to study, since there are a lot of languages and each language has its own scope of application. According to the results of the survey, respondents stated that they understand exactly which programming languages they should use for their tasks, the rest found it difficult to answer the question posed.

Goal

Make it easier to find a programming language to implement tasks in your IT project using a test.

Tasks

identify

To identify the need to create a test using a survey of TSU students.

explore

Study the current list of programming languages and the scope of its application.

Systematize

Systematize information on programming languages and identify the range of tasks that each of them can solve

create

Create a test based on an existing platform.
